Описание компоненты cis1-core-bash

Данная компонента является Ядром cis1.
Компонента разработана на скриптовом языке bash.
Компонента полностью совместима с ОС семейства Linux, и возможно ее можно запустить на ОС Windows, если установить на нее определенные расширения, поддерживающие POSIX.

Компонента полностью удовлетворяет общим требования к ядру cis1 и реализует его бизнес-логику. Общее описание можно найти в репозитарии cis1-docs, файл overview.html.

Передача параметров в дочернюю задачу

$cis_base_dir/setparam param_name param_value


Example:
$cis_base_dir/core/setparam "prm2" "$prepared_prm2"
$cis_base_dir/core/setparam "prm3" "$prepared_prm3"
$cis_base_dir/core/startjob internal/core_test_sub_startjob_withargs

Механизм глобальных переменных сессии


Запись переменной в сессию:
$cis_base_dir/core/setvalue name value

Example:

$cis_base_dir/core/setvalue "prm1" "$prm1"
$cis_base_dir/core/setvalue "prm2" "$prm2"
$cis_base_dir/core/setvalue "prm3" "$prm3"

Чтение переменной:
$cis_base_dir/core/getvalue name

Example:

prm1=$($cis_base_dir/core/getvalue prm1)
prm2=$($cis_base_dir/core/getvalue prm2)
prm3=$($cis_base_dir/core/getvalue prm3)
